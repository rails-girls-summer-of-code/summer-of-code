Student applications for this year's Rails Girls Summer of Code will be opened in March 2015.

In the meantime, check out these sections for information about the application process, requirements for participating in the Rails Girls Summer of Code as well as tips and suggestions on how to organize yourselves should you be granted a sholarship.

  • Application Guide: Information about all the requirements for a complete application.

  • What to expect: Activities during the program and suggestions on how to meet the requirements for participating in the Summer of Code. This section also gives you an idea of what a Summer of Code will look like.

  • Conferences: Join the community, talk about your Summer of Code and get inspired. We will again try and provide conference tickets for many events worldwide.
